About the Pasta House Bookmarklet Index (BMI)

Welcome to Pasta House, where you can find quality bookmarklets, both user-submitted and created.

What is a bookmarklet?

A bookmarklet is a bookmark that can run code.

Why bookmarklets?

Bookmarklets can help you do anything from automating daily tasks, such as blocking YouTube ads, to pulling a prank on friends, such as blurring the page.

How do I install a bookmarklet?

Just create a new bookmark and set the URL to the code beginning with "javascript:". To import all bookmarklets into your browser, you can find instructions here.
